# 🗓️ 6-Month Roadmap (180 Days Plan)

---

## 📅 Phase 1 (Day 1–30) — Fundamentals

### Topics:

* Arrays
* Strings
* Basic Math
* Time & Space Complexity

### Daily Plan:

* ⏱️ 2–3 hours
* 📌 5 problems/day

### Focus:

* Brute force → Better → Optimal
* Learn complexity analysis

---

## 📅 Phase 2 (Day 31–60) — Core Data Structures

### Topics:

* Linked List
* Stack
* Queue
* Recursion

### Focus:

* Dry run every problem
* Understand recursion tree

---

## 📅 Phase 3 (Day 61–90) — Patterns + Searching

### Topics:

* Sliding Window
* Two Pointers
* Binary Search

### Focus:

* Recognize patterns instantly
* Solve without hints

---

## 📅 Phase 4 (Day 91–120) — Trees & Heaps

### Topics:

* Binary Trees
* BST
* Heap / Priority Queue

### Focus:

* Traversals (DFS, BFS)
* Tree-based recursion

---

## 📅 Phase 5 (Day 121–150) — Graph + Greedy

### Topics:

* Graphs (DFS, BFS)
* Shortest Path
* Greedy Algorithms

---

## 📅 Phase 6 (Day 151–180) — Dynamic Programming

### Topics:

* DP Basics
* Memoization
* Tabulation
* Advanced DP Patterns

# 🧩 How to Solve Any DSA Problem

### Step-by-Step Approach:

1. **Understand problem**
2. **Think brute force**
3. **Optimize step by step**
4. **Write pseudo-code**
5. **Code**
6. **Dry run**
7. **Optimize further**
